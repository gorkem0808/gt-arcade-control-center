# Kurulum

## 1. GitHub'a yükleme

1. Yeni repo aç.
2. Bu klasörün içindeki dosyaları yükle.
3. Actions sekmesine gir.
4. Yeşil tik olunca Artifacts bölümünden UF2 ZIP'i indir.

## 2. Pico yükleme

- P1 için: `gt_arcade_p1_absolute_mouse.uf2`
- P2 için: `gt_arcade_p2_absolute_mouse.uf2`
- Tuş Pico için: `gt_arcade_button_keyboard.uf2`

BOOTSEL basılı tut, USB tak, RPI-RP2 diskine UF2 dosyasını kopyala.

## 3. PC programı

`pc_app/GT_ARCADE_CONTROL_CENTER.bat` çalıştır.

Önce 3 Pico'yu tak, sonra programda **Cihazları Tara** butonuna bas.
