# Pin Listesi

## Pico 1 / Pico 2 - Absolute Mouse

| Görev | Pin |
|---|---|
| X potans orta uç | GP26 |
| Y potans orta uç | GP27 |
| Sistem aktif / pasif | GP20 -> GND olunca aktif |

Potans dış uçları: 3V3 ve GND.

GP20 basılı değilken Pico mouse raporu göndermez. Böylece normal PC mouse serbest kalır.
